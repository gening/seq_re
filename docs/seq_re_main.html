seq_re.seq_re_main module¶

Main module of sequence regular express pattern¶

Compile the SEQ RE pattern into an ordinary regular expression (RE) pattern. And every dimension of the sequence is encoded and arranged into a string, which is scanned by the RE pattern using search() or findall() functions.

class seq_re.seq_re_main.SeqRegex(len_tuple)[source]¶

Bases: object

Encode each element of the pattern and sequence into a single char.

And then form the pattern and sequence into a linear string, which are suitable for using the ordinary regular expression (RE) to match.

Finally, the match result will be located in the original sequence.
